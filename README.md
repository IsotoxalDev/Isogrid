# Isogrid

Isogrid is an infinite digital workspace for creative collaboration. It provides a simple and intuitive canvas for you to organize your ideas, create mood boards, and visualize your thoughts.

## Features

- **Infinite Canvas:** Pan and zoom around an endless creative space.
- **Add Content:** Add text, images (by pasting), and create nested boards.
- **Organize with Arrows:** Connect your ideas with adjustable arrows.
- **Customizable Environment:** Adjust the grid style, colors, and vignette to create your preferred workspace.
- **Nested Boards:** Create boards within boards for deeper organization.

## Getting Started

To get started, simply run the development server:

```bash
npm run dev
```

Then open your browser to [http://localhost:9002](http://localhost:9002) to start using Isogrid.
