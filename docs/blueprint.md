# **App Name**: CanvasCraft

## Core Features:

- Infinite Canvas: Provides a limitless digital workspace for arranging and connecting items.
- Drag-and-Drop Items: Enables users to drag and drop text, images, or other boards onto the canvas.
- Right-Click Context Menu: A context menu appears upon right-clicking, offering options to add new elements (text, images, boards) or connect elements via arrows.
- Image Copy-Paste: Allows direct pasting of images from the clipboard onto the canvas.
- Arrow Drawing: Draw arrows to connect items via a right-click action to show relationships between the board elements.
- Toggleable Grid: Allows the user to make the backgroud grid visible and invisible.

## Style Guidelines:

- Primary color: c5d86d, main accent color in dark mode, following isotoxal colorscheme with 151515 and shades.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral feel.
- Simple, clean icons for tools and actions in the context menu and sidebar.
- Clean and intuitive layout, ensuring elements are easily accessible.
- Subtle animations for item dragging and placement to enhance user feedback.
